Dashfy Registry

The Dashfy extension registry. Extensions are resolved from here by the Dashfy CLI.

  • /r/index.json — catalog of available extensions
  • /r/<name>.json — a single extension item (e.g. /r/github.json)
  • /schema/registry-item.json — item JSON Schema
  • /schema/registry.json — index JSON Schema

Install an extension: npx dashfy add @getdashfy/github
